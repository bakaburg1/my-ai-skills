# Guides and Labels

Read this file when the task is about axes, legends, captions, titles,
annotations, or faceting details.

## Core Rules

- Make titles and captions state the analytical point of the plot.
`labs(title = "Revenue rose after the intervention")`
- Keep legends readable; remove them only when the encoding is obvious.
- Use labels and annotations to explain the plot, not to restate the
  obvious.

## Avoid

- over-annotating simple plots
- forcing dense legends to carry too much meaning
